Сделай мне продающий лендинг для продажи локальной лицензии конструктора Интеграм для малого и среднего бизнеса. Клиент должен захотеть автоматизировать и обезопасить свой бизнес-процесс, который, как правило, ведется в excel. Типичные примеры: учет затрат по проектам, ресурсное планирование по проектам, планирование производства, расчет бюджетов, планирование отпусков и ведение табелей. Поскольку клиент работает с непубличными данными, он хочет разместить конструктор у себя - на своих серверах, для чего покупает локальную лицензию.
